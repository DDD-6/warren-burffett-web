# 서비스 소개
지금 내가 얼마를 버는지 알고 싶으신가요? 
퇴근 욕구가 드는 지금 내가 벌고 있는 돈이 보인다는 것은 일에 추진력을 줍니다. 
가계부를 정리하면 과거의 나 왜 그랬니라는 생각이 드시나요?
미래의 나도 결국 나라는 것을 깨달으시나요?
절약 챌린지를 통해 미래의 나를 구해줄 나를 만들어 드립니다

# 개발 스택
- nextjs, typescript
- react-query
- aws amplify
